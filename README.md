# Gron

[![GitHub go.mod Go version](https://img.shields.io/github/go-mod/go-version/alnovi/gron)](https://go.dev/dl/)
[![GitHub License](https://img.shields.io/github/license/alnovi/gron)](https://github.com/alnovi/gron/blob/master/LICENSE.md)

**Gron** — парсер cron выражений для golang.

**Cron** — планировщик задач в Unix-подобных системах, который позволяет запускать команды и скрипты по расписанию.
Его выражения состоят из пяти полей, разделённых пробелами, каждое из которых определяет определённый временной
интервал.

```
┌────────── minute (0 - 59)  
| ┌──────── hour (0 - 23)  
| | ┌────── day of month (1 - 31)  
| | | ┌──── month (1 - 12)  
| | | | ┌── day of week (0 - 6, sunday=0)  
| | | | |  
* * * * *   command  
```
## Поддерживаемые символы

| Символ | Обозначение           |   Пример   | Эквивалент                           |
|:------:|-----------------------|:----------:|--------------------------------------|
|   *    | Любое значение        | * * * * *  | Каждую минуту                        |
|   -    | Диапазон значений     | 1-10 * * * | Минуты с 1 по 10                     |
|   ,    | Список значений       | 1,10 * * * | На 1 и 10 минутах                    |
|   /    | Значения шага         | */10 * * * | Каждые 10 минут                      |
|   L    | Последний день месяца | 0 0 L * *  | В 00:00 каждый последний день месяца |
|   W    | Рабочие дни           | 0 0 W * *  | В 00:00 с Пн по Пт                   |

## Установка

```sh
go get github.com/alnovi/gron
```

## Использование

```go
package main

import (
	"fmt"
	"time"
	"github.com/alnovi/gron"
)

func main() {
	next, _ := gron.NextTime("* * * * *")
	fmt.Print(next.Format(time.DateTime))
}

// out: "2026-01-01 00:01:00"
```

## Некоторые из распространённых выражений

|    Выражение     | Описание                                     |
|:----------------:|----------------------------------------------|
|   `0 0 * * *`    | запуск задачи один раз в день в полночь      |
|  `*/15 * * * *`  | запуск каждые 15 минут                       |
|  `0 */2 * * *`   | запуск каждые 2 часа                         |
| `0 9-17 * * 1-5` | запуск каждый час с 9:00 до 17:00 с пн по пт |
|   `0 0 1 * *`    | запуск в полночь первого дня каждого месяца  |
